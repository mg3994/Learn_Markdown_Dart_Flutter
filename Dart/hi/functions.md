# डार्ट फ़ंक्शन्स

डार्ट में, फ़ंक्शन्स वस्तुएं होती हैं और इनका एक प्रकार, `Function`, होता है। इसका मतलब है कि फ़ंक्शन्स को चर के रूप में निर्धारित किया जा सकता है या अन्य फ़ंक्शन्स को तर्क के रूप में पास किया जा सकता है। आप डार्ट कक्ष के एक उदाहरण को भी कॉल कर सकते हैं जैसे कि यह एक फ़ंक्शन हो।

यहाँ डार्ट में विभिन्न प्रकार के फ़ंक्शन्स हैं:

1. **टॉप-लेवल फ़ंक्शन्स**: ये फ़ंक्शन्स फ़ाइल के टॉप-लेवल पर परिभाषित होते हैं, किसी भी कक्ष के बाहर।

    ```
    void main() {
      print('नमस्ते, दुनिया!');
    }
    ```

2. **स्थिर विधियाँ**: ये फ़ंक्शन्स एक कक्ष के साथ संबद्ध होती हैं लेकिन उस कक्ष के एक उदाहरण के साथ नहीं।

    ```
    class MyClass {
      static void myStaticFunction() {
        print('नमस्ते, दुनिया!');
      }
    }
    ```

3. **उदाहरण विधियाँ**: ये फ़ंक्शन्स एक कक्ष के एक उदाहरण के साथ संबद्ध होती हैं।

    ```
    class MyClass {
      void myInstanceFunction() {
        print('नमस्ते, दुनिया!');
      }
    }
    ```

4. **अनाम फ़ंक्शन्स**: ये अनाम फ़ंक्शन्स होती हैं जिनका उपयोग एक विधि कॉल में एक पैरामीटर के रूप में किया जा सकता है।

    ```
    var list = ['सेब', 'केले', 'संतरे'];
    list.forEach((item) {
      print('${list.indexOf(item)}: $item');
    });
    ```

प्रत्येक प्रकार का फ़ंक्शन अपना उपयोग मामला होता है और आवश्यकता के आधार पर उपयोग किया जा सकता है।
