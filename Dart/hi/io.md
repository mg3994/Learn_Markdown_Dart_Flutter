#### io: डार्ट में इनपुट/आउटपुट ऑपरेशन

डार्ट में, इनपुट/आउटपुट (I/O) ऑपरेशन फ़ाइलों, स्ट्रीमों, और अन्य इनपुट/आउटपुट उपकरणों के साथ बातचीत के लिए महत्वपूर्ण हैं। डार्ट विभिन्न I/O ऑपरेशन करने के लिए योग्य पुस्तकालयों और API प्रदान करता है।

##### फ़ाइल हैंडलिंग
डार्ट फ़ाइलों से पढ़ने और लिखने जैसे फ़ाइल हैंडलिंग ऑपरेशन का समर्थन करता है। `dart:io` पुस्तकालय फ़ाइलों और निर्देशिकाओं के साथ काम करने के लिए `File` और `Directory` जैसी कक्षाएं प्रदान करता है। डेवलपर इन कक्षाओं का उपयोग करके पढ़ने, लिखने, कॉपी करने, और फ़ाइलों को हटाने जैसे ऑपरेशन कर सकते हैं।

##### स्ट्रीमों
डार्ट असिंक्रोनस I/O ऑपरेशन को हैंडल करने के लिए स्ट्रीम की अवधारणा का लाभ उठाता है। स्ट्रीम डेटा के अनन्त क्रमों को प्रतिनिधित करते हैं जो निरंतर उत्पन्न या सेवित होते हैं। डार्ट `Stream` कक्षा और विभिन्न स्ट्रीम ट्रांसफ़ॉर्मर असिंक्रोनस डेटा के साथ काम करने के लिए प्रदान करता है।

##### मानक इनपुट/आउटपुट
डार्ट मानक इनपुट और आउटपुट स्ट्रीम के साथ बातचीत करने की अनुमति देता है। डेवलपर्स `stdin.readLineSync()` जैसी विधियों का उपयोग कर सकते हैं ताकि कॉन्सोल से इनपुट पढ़ सकें और `stdout.write()` का उपयोग कर सकते हैं ताकि कॉन्सोल पर आउटपुट लिख सकें।

##### नेटवर्क ऑपरेशन
डार्ट अपने `dart:io` पुस्तकालय के माध्यम से नेटवर्क I/O ऑपरेशन को संभव बनाता है। डेवलपर्स डार्ट के नेटवर्किंग API का उपयोग करके क्लाइंट और सर्वर अनुप्रयोग बना सकते हैं, HTTP अनुरोध कर सकते हैं, और डार्ट के नेटवर्क संचार को हैंडल कर सकते हैं।

डार्ट की I/O क्षमताओं का उपयोग करके, डेवलपर्स शक्तिशाली अनुप्रयोग बना सकते हैं जो फ़ाइलों, स्ट्रीमों, कॉन्सोल, और नेटवर्क संसाधनों के साथ प्रभावी ढंग से बातचीत करते हैं।

डार्ट में इनपुट/आउटपुट ऑपरेशन के बारे में अधिक विस्तृत जानकारी के लिए, आधिकारिक डार्ट दस्तावेज़ और गाइड्स का संदर्भ लें।

# डार्ट में इनपुट/आउटपुट ऑपरेशन

इनपुट/आउटपुट (I/O) ऑपरेशन हर किसी प्रोग्रामिंग भाषा के लिए महत्वपूर्ण होते हैं क्योंकि ये बाहरी दुनिया के साथ संचार को संभव बनाते हैं, जैसे फ़ाइलों से डेटा पढ़ना, उपयोगकर्ता से इनपुट प्राप्त करना, और कॉन्सोल पर आउटपुट प्रिंट करना। डार्ट में, इनपुट/आउटपुट ऑपरेशन को कार्यात्मकता से करने के कई तरीके उपलब्ध हैं।

## कॉन्सोल से इनपुट पढ़ना

डार्ट में कॉन्सोल से इनपुट पढ़ने के लिए, आप `dart:io` पेयलिब्रेरी द्वारा प्रदान की गई `stdin` स्ट्रीम का उपयोग कर सकते हैं। यहां एक उदाहरण है कि कैसे कॉन्सोल से एक पंक्ति टेक्स्ट पढ़ी जाए:

```
import 'dart:io';

void main() {
  print('अपना नाम दर्ज करें: ');
  String name = stdin.readLineSync();
  print('नमस्ते, $name!');
}
```

इस उदाहरण में, `stdin.readLineSync()` का उपयोग किया गया है ताकि कॉन्सोल से उपयोगकर्ता द्वारा दर्ज की गई एक पंक्ति पाई जा सके।

## कॉन्सोल पर आउटपुट लिखना

डार्ट में कॉन्सोल पर आउटपुट लिखने के लिए, आप `dart:io` पेयलिब्रेरी द्वारा प्रदान की गई `stdout` ऑब्जेक्ट का उपयोग कर सकते हैं। यहां एक उदाहरण है कि कैसे कॉन्सोल पर आउटपुट प्रिंट किया जाए:

```
import 'dart:io';

void main() {
  stdout.write('अपना नाम दर्ज करें: ');
  String name = stdin.readLineSync();
  stdout.write('अपनी उम्र दर्ज करें: ');
  int age = int.parse(stdin.readLineSync());
  
  print('नमस्ते, $name! आप $age वर्षीय हैं।');
}
```

इस उदाहरण में, `stdout.write()` का उपयोग किया गया है ताकि पाठ को कॉन्सोल पर नई लाइन वर्तन न हो।

## फाइलों के साथ काम करना

डार्ट में फ़ाइलों और निर्देशिकाओं के साथ काम करने के लिए क्लासेज `dart:io` पेयलिब्रेरी में उपलब्ध हैं। यहां एक उदाहरण है कि कैसे एक पाठ फ़ाइल की सामग्री को पढ़ा जाए:

```
import 'dart:io';

void main() {
  File file = File('example.txt');
  if (file.existsSync()) {
    String contents = file.readAsStringSync();
    print('फ़ाइल की सामग्री: $contents');
  } else {
    print('फ़ाइल नहीं मिली।');
  }
}
```

इस उदाहरण में, `File` का उपयोग एक फ़ाइल को प्रतिनिधित करने के लिए किया गया है, और `readAsStringSync()` का उपयोग फ़ाइल की सामग्री को एक स्ट्रिंग के रूप में पढ़ने के लिए किया गया है।

## असिंक्रोनस इनपुट/आउटपुट ऑपरेशन

डार्ट भी फ्यूचर्स और एसिंक/वेट सिंटेक्स का समर्थन करता है जो कि असिंक्रोनस इनपुट/आउटपुट ऑपरेशन को संभव बनाता है। यह आपको अपने प्रोग्राम के निष्पादन को ब्लॉक न करते हुए इनपुट/आउटपुट ऑपरेशन करने की अनुमति देता है। यहां एक उदाहरण है एक फ़ाइल को असिंक्रोनस रूप से पढ़ने का:

```
import 'dart:io';

void main() async {
  try {
    File file = File('example.txt');
    String contents = await file.readAsString();
    print('फ़ाइल की सामग्री: $contents');
  } catch (e) {
    print('फ़ाइल पढ़ने में त्रुटि: $e');
  }
}
```

इस उदाहरण में, `await file.readAsString()` का उपयोग फ़ाइल की सामग्री को असिंक्रोनस रूप से पढ़ने के लिए किया गया है।

## निष्कर्षण

डार्ट में, इनपुट/आउटपुट ऑपरेशन को सरल और प्रभावी ढंग से करना संभव है, धन्यवाद बनाये रखने के लिए निर्मित पुस्तकालयों और असिंक्रोनस समर्थन का। चाहे आपको कॉन्सोल के साथ या फ़ाइलों के साथ काम करना हो, डार्ट वे आवश्यक उपकरण प्रदान करता है जो इनपुट/आउटपुट ऑपरेशनों को प्रभावी ढंग से संभालने की सहायता करते हैं।
